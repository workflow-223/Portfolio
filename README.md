# Main Portfolio

Personal portfolio site built with React 19, TypeScript, Tailwind CSS 4, and Framer Motion.

## Sections

- Hero with typewriter animation and particle network
- About with bio and highlights
- Experience timeline
- Projects grid with filtering
- Skills with animated progress bars
- Education with coursework tags
- Contact form with social links

## Deployment

Deployed to GitHub Pages at:
`https://workflow-223.github.io/main-portfolio/`

## Local Setup

```bash
npm install
npm run dev
```
